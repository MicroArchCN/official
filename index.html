通过标准化的接口与 SDK，任何系统都可以在分钟级接入 MicroArch AI 能力，无需了解底层模型细节。

申请接入
microarch-sdk — demo
# 初始化 MicroArch AI 客户端
from microarch import AIClient
 
client = AIClient(
  api_key="ma_live_xxxxxx",
  workspace="acme-corp"
)
 
# 调用知识库增强对话
response = client.chat(
  query="分析本季度销售趋势",
  knowledge_base="sales-reports"
)
 
# 输出结构化结果
print(response.answer)
